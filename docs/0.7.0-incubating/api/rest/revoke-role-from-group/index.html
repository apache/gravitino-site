Revoke roles to a group

PUT 
/metalakes/:metalake/permissions/groups/:group/revoke

Revoke roles to a group

Request​

Path Parameters

    metalake stringrequired

    The name of the metalake

    group stringrequired

    The name of the group

  • application/json

Body

    roleNames string[]required

    The role names need to revoke

Responses​

  • 200
  • 400
  • 404
  • 5xx

Returns the revoked group object

  • application/vnd.gravitino.v1+json

    The name of the group

    roles string[]

    The roles of the group

    Audit information for a Gravitino resource.

    creator string

    The user who created the resource

    createTime date-time

    The time the resource was created

    lastModifier string

    The user who last modified the resource

    lastModifiedTime date-time

    The time the resource was last modified
