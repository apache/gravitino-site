class BaseSchemaCatalog(gravitino.dto.catalog_dto.CatalogDTO, gravitino.api.supports_schemas.SupportsSchemas)
    BaseSchemaCatalog(catalog_namespace: gravitino.namespace.Namespace, name: str = None, catalog_type: gravitino.api.catalog.Catalog.Type = &lt;Type.UNSUPPORTED: 'unsupported'&gt;, provider: str = None, comment: str = None, properties: Dict[str, str] = None, audit: gravitino.dto.audit_dto.AuditDTO = None, rest_client: gravitino.utils.http_client.HTTPClient = None)
 
BaseSchemaCatalog is the base abstract class for all the catalog with schema. It provides the
common methods for managing schemas in a catalog. With BaseSchemaCatalog, users can list,
create, load, alter and drop a schema with specified identifier.
