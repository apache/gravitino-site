class GravitinoClient(gravitino.client.gravitino_client_base.GravitinoClientBase)
    GravitinoClient(uri: str, metalake_name: str, check_version: bool = True, auth_data_provider: gravitino.auth.auth_data_provider.AuthDataProvider = None)
 
Gravitino Client for a user to interact with the Gravitino API, allowing the client to list,
load, create, and alter Catalog.
 
It uses an underlying {@link RESTClient} to send HTTP requests and receive responses from the API.
